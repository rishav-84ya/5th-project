EDU-SHARE is a web-based academic resource sharing platform designed to connect students and 
teachers across different universities and colleges. It enables users to easily upload, access, 
and download study materials such as notes, papers, and academic resources in a structured and secure environment.

The platform promotes free and open knowledge sharing, organizing materials by University, 
Department, Course, and Subject, making it easy to find relevant content. EDU-SHARE encourages collaboration beyond institutional boundaries and helps build a shared academic community.

Tech that are used :--
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Security: Session management & prepared statements
